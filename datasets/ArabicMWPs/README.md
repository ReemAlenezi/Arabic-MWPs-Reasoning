# ArabicMWPs Dataset

ArabicMWPs is an Arabic mathematical word problem dataset constructed by translating two existing English datasets: AddSub and MultiArith.

## Description

* Language: Arabic
* Domain: Mathematical word problems
* Size: 725 questions

  * 265 from AddSub
  * 460 from MultiArith

## Construction

The dataset was created by translating problems from the original English datasets into Arabic. The translation preserves the mathematical meaning and structure of the problems.

## Format

Each example includes:

* Question
* Final answer

## Purpose

ArabicMWPs is designed to evaluate basic arithmetic reasoning capabilities of language models in Arabic.

## Notes

This dataset is derived from AddSub (Hosseini et al., 2014) and MultiArith (Roy & Roth, 2016), and adapted for Arabic NLP research.
