
# ArabicMWPs Dataset

ArabicMWPs is an Arabic mathematical word problem dataset constructed by translating two English datasets: AddSub and MultiArith.

## Description
- Language: Arabic
- Domain: Mathematical word problems
- Size: 725 questions
  - 265 from AddSub
  - 460 from MultiArith

## Format
Each example includes:
- Question
- Final answer

## Purpose
This dataset is designed to evaluate arithmetic reasoning capabilities of large language models in Arabic.

## Notes
This dataset is derived from AddSub (Hosseini et al., 2014) and MultiArith (Roy & Roth, 2016).
