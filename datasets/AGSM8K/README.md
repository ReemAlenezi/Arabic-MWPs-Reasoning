# AGSM8K Dataset

AGSM8K is an Arabic mathematical word problem dataset introduced in this work. It is translated from the original GSM8K dataset (Cobbe et al., 2021) and adapted to support Arabic mathematical reasoning tasks.

## Description
- Language: Arabic
- Domain: Grade-school mathematical reasoning
- Size:
  - Training set: 7,473 examples
  - Test set: 1,319 examples

## Construction
The dataset was generated by translating GSM8K problems into Modern Standard Arabic using ChatGPT. All translations were manually reviewed to ensure:
- Mathematical correctness  
- Linguistic accuracy  
- Consistency with the original problem intent  

## Format
Each example includes:
- Question  
- Reasoning steps  
- Final answer  

## Purpose
AGSM8K is designed to evaluate the ability of large language models to solve Arabic mathematical word problems and generate coherent reasoning steps.

## Notes
This dataset is a translated and curated version of GSM8K and is intended for research purposes.
