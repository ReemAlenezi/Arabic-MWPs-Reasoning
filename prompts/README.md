# Prompts

This directory contains the prompt templates used in the experiments.

## Languages
- English
- Arabic

## Prompting Techniques
- Zero-shot Chain-of-Thought (CoT)
- Manual Chain-of-Thought (CoT)
- Self-consistency : is implemented by sampling multiple outputs using the same prompt (Manual CoT), and selecting the final answer based on majority voting.

These prompts guide large language models to generate step-by-step reasoning for solving mathematical word problems.
